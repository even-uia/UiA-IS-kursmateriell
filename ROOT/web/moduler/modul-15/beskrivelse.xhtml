Modul 15: Web-applikasjoner i Java

I denne modulen skal vi se på hvordan vi kan lage web-applikasjoner i java, ved å bruke servlets, som veldig kort beskrevet er java-objekter som kan lage web-sider (html dokumenter) på sparket

Læringsmål:

Du har fullført modulen når du:

  • Kan lage servets som tar i mot data fra en form, og gjør noe fornuftig med dem.
  • Kan skrive servlets som (lager sider som) viser fram resultatet av bereginger/prosessering av data
  • Kan forklare hvordan en web-server håndterer requests (spørringer fra en browser), og hvordan du får opp en side i browseren som er laget av en servlet

Oppgave

Installer Apache Tomcat (tomcat.apache.org)

Opprett ett nytt web-application prosjekt. Lag minst en form, og en servlet som behandler den.

Ta med prosjektet til intervjuet med hjelpelærer eller foreleser

Godkjenning

Godkjennes av hjelpelærer eller foreleser i intervju.
