Modul 10: Arv og subklasser II

Denne modulen tar opp mer avanserte sider ved arv og subklasser. Det kan f.eks. være ønskelig, og noen ganger helt nødvendig, å kunne omdefinere metoder som er arvet fra superklassen. Det gir oss et par nye problemstillinger: Hvordan vet vi at det er den riktige metoden som blir kalt? I noen tilfeller trenger vi tilgang til både den omdefinerte, og den opprinnelige metoden. Hva gjør vi da? Denne modulen gir deg svar på disse spørsmålene

Læringsmål:

Du har fullført modulen når du:

  • Kan omdefinere metoder som er arvet fra en superklasse
  • Kan kalle den opprinnelige metoden når det er nødvendig
  • Kan kalle constructor'en til superklassen.
  • Kan forklare forskjellen på statisk og dynamisk type
  • Kan forklare begrepet polymorfisme
  • Kan forklare hvordan runtimesystemet i Java finner riktig metode, når et metodekall skal utføres

Oppgave

Fortsett med prosjektet ditt (eller start et nytt), og ta i bruk det du har lært i denne modulen, spesielt overriding og polymofisme

Godkjenning

Godkjennes av foreleser eller hjelpelærer i intervju
